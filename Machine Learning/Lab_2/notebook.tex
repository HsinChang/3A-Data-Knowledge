    So we can see that in the iris dataset, the number of the three classes
(0,1,2) are all 50. So, In each step of the cross-validation, we suppose
that we take \(a\) samples from class A, \(b\) from class B, \(c\) from
class C for the test. So we have \(a+b+c = \frac{150}{k}\), let us
suppose \(a>b>c\), so \(c\) will be the majority class in the training
as less \(c\) is taken into test set. And \(c<\frac{50}{k}\), so we get
\[accurary = \frac{c \times k}{150}<\frac{50}{150}<\frac{1}{3}\] So for
the iris dataset with the majority class classifier, the accuracy cannot
be larger than \(\frac{1}{3}\), or for each dataset, the accuracy cannot
be larger than \(\frac{1}{No. classes}\)
